### Update `start_instance`

The `:id` option is now required. Fetch it and pass to the runner:

```elixir
def start_instance(workflow, context, opts \\ [])
    when is_atom(workflow) and is_map(context) do
  id = Keyword.fetch!(opts, :id)  # required now

  {version, resolved_module} = ... # existing version resolution

  telemetry_metadata = Keyword.get(opts, :telemetry_metadata, %{})

  @hephaestus_runner_module.start_instance(
    resolved_module,
    context,
    Keyword.merge(runner_opts(),
      id: id,  # NEW
      telemetry_metadata: telemetry_metadata,
      workflow_version: version
    )
  )
end
```
